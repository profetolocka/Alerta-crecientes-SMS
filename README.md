# Alerta-crecientes-SMS


 Sistema de alerta de crecientes diseñado como parte de un proyecto escolar junto a mis alumnos del IPETyM 61 en el año 2017.
 
 Está basado en un Arduino UNO y reporta novedades a través de un shield celular utilizando mensajes SMS.

 Se alimenta desde la red eléctrica y un panel solar.
